================= Linux Basic Commands ======= #1 ls There are variations you can use with the ls command: A) ls -R will list all the files in the sub-directories as well B) ls -a will show the hidden files C) ls -al will list the files and directories with detailed information like the permissions, size, owner, etc. #2 cat Here are other ways to use the cat command: A) cat > filename ==> creates a new file B) cat filename1 filename2>filename3 ==> joins two files (1 and 2) and stores the output of them in a new file (3) C) cat filename | tr a-z A-Z >output.txt ==> to convert a file to upper or lower case use, Note: After Creating a new file and added some text for exiting and saving the data do this A) ctrl + B // for close or ctrl + C B) ctrl + D // to save the file #3
